    QuoteSyncHandler

    QuoteSyncHandler class that contains all the logic that uses the QuoteSyncUtil. This class exposes methods to sync the Opportunity, Quote, OpportunityLineItem and QuoteLineItem objects and should ideally be called from each object's trigger
    Code adapted from QuoteSync AppExchange package, moved to single handler class
